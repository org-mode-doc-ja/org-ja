* Org-ja とは何ですか？

このプロジェクトは，英語で公開されている org-mode のマニュアルを日本語化しています．単純に翻訳するだけではなく「永続的に翻訳内容をアップデートできること」を条件としています．そのため，たくさんの有志の方に支えられて共有資産としてオープンに運用されています．
